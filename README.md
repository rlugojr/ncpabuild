NCPA Build
=========

A package of scripts and tarballs for installing an NCPA build box.


Windows builds
---------
To build on windows you must install ActivePython 2.7 and the following pip components for python:
<pre>
cx-Freeze
cx-Logging
cx-PyGenLib
docutils
Flask
gevent
gevent-websocket
greenlet
Jinja2
MarkupSafe
psutil
Pygments
pyOpenSSL
pypm
pythonselect
pywin32
requests
setuptools
Sphinx
sphinx-bootstrap-theme
virtualenv
Werkzeug
wsgiref
</pre>
